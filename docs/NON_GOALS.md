# Non-Goals

Explicit non-goals prevent drift.

- No dashboard home screen.
- No AI chat as default UI.
- No Slack in v0.
- No Gmail in v0.
- No Calendar in v0 unless after local core.
- No cloud sync in v0.
- No team collaboration in v0.
- No real-time multiplayer.
- No mobile app before Mac app feels excellent.
- No proprietary-only note format.
- No account requirement for local use.
- No autonomous sending.
- No hidden cloud database.
- No productivity score.
- No streaks.
- No gamification.
- No red notification badges.
- No broad plugin system in v0.
- No arbitrary dynamic block programming in v0.
- No new dependencies without a decision record.
