# Daymark

Daymark is a high-craft, local-first macOS workspace centered on today's Markdown note.

This repository is currently in the governance and scaffold stage. The goal is to create durable instructions, module boundaries, tests, and a native Swift structure before product features begin.

## Current Milestone

Governance and scaffold. Milestone 0, the taste prototype, comes next.

## Build

```bash
swift build
swift test
swift run daymark doctor
```

The package includes a SwiftUI app shell target named `Daymark` and a future CLI target named `daymark`.
